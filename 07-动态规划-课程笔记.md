## 零、DP 思维框架

### 什么是 DP？

把一个大问题拆成**重叠的子问题**，每个子问题只算一次，保存结果供后续使用。

### 做 DP 题的思路（yxc 套路）

1. **状态表示** `f[i][j]` — 考虑什么？表示什么？
   - **集合**：f[i][j] 代表哪一类方案的集合？
   - **属性**：max / min / count？
2. **状态计算** — 集合怎么划分？转移方程是什么？
3. **初始化** — 边界条件
4. **遍历顺序** — 确保计算 f[i][j] 时，依赖的状态已经算好

## 一、背包问题（DP 入门经典）

### 1.1 01 背包（AcWing 2）

**问题**：N 件物品，每件只能选一次。体积 vi，价值 wi。背包容量 V。求最大价值。

**状态表示**：
- `f[i][j]` = 只考虑前 i 件物品，总体积 ≤ j 的最大价值

**状态计算**（第 i 件选不选）：
```
f[i][j] = max(f[i-1][j],              // 不选第 i 件
              f[i-1][j-vi] + wi)       // 选第 i 件
```

**空间优化**：因为只依赖上一层，可以压成一维。**但必须倒序遍历 j**（否则同一件物品会被选多次）。

```cpp
for (int i = 1; i <= n; i++)
    for (int j = V; j >= v[i]; j--)   // ⚠️ 倒序！
        f[j] = max(f[j], f[j - v[i]] + w[i]);
```

**为什么倒序？** 如果正序，`f[j-v[i]]` 可能已经是 i 这轮更新过的（相当于第 i 件选了多次）。倒序保证 `f[j-v[i]]` 还是上一轮的值。

---

### 1.2 完全背包（AcWing 3）

**问题**：每件物品可以**选无限次**。

**朴素转移**：
```
f[i][j] = max(f[i-1][j], f[i-1][j-v]+w, f[i-1][j-2v]+2w, ...)
```

**优化观察**：
```
f[i][j]   = max(f[i-1][j], f[i-1][j-v]+w, f[i-1][j-2v]+2w, ...)
f[i][j-v] = max(           f[i-1][j-v],   f[i-1][j-2v]+w, ...)
```
所以 `f[i][j] = max(f[i-1][j], f[i][j-v] + w)`

**空间优化**：压成一维，**正序遍历 j**（跟 01 背包相反！因为就是要用这一轮更新过的值）。

```cpp
for (int i = 1; i <= n; i++)
    for (int j = v[i]; j <= V; j++)    // ⚠️ 正序！
        f[j] = max(f[j], f[j - v[i]] + w[i]);
```

---

### 1.3 多重背包（AcWing 4, 5）

**问题**：第 i 件物品最多选 si 次。

**朴素做法**：枚举选 0~si 个，O(NVS)。

**二进制优化**（AcWing 5）：把 si 个拆成若干"打包"的物品：1, 2, 4, 8, ..., 剩余。这样用 01 背包处理，复杂度 O(NV log S)。

```cpp
// 把第 i 件物品拆成 log(si) 个打包物品
int k = 1;
while (k <= s) {
    goods.push_back({k * v, k * w});
    s -= k;
    k *= 2;
}
if (s > 0) goods.push_back({s * v, s * w});
// 然后对 goods 做 01 背包
```

---

### 1.4 分组背包（AcWing 9）

**问题**：物品分成若干组，每组最多选一件。

**转移**：枚举每组选哪件（或不选）。

```cpp
for (int i = 1; i <= n; i++)           // 枚举每组
    for (int j = V; j >= 0; j--)       // 倒序（01 背包思路）
        for (int k = 0; k < s[i]; k++) // 枚举组内选哪件
            if (j >= v[i][k])
                f[j] = max(f[j], f[j - v[i][k]] + w[i][k]);
```

### 背包问题速查

| 类型 | 特点 | 一维遍历顺序 | 复杂度 |
|------|------|-------------|--------|
| 01 背包 | 每件选一次 | j 倒序 | O(NV) |
| 完全背包 | 每件无限次 | j 正序 | O(NV) |
| 多重背包 | 每件 si 次 | 二进制拆分→01 | O(NV log S) |
| 分组背包 | 每组选一件 | j 倒序 + 枚举组内 | O(NVK) |

## 二、线性 DP

### 2.1 数字三角形（AcWing 898）

```
从顶到底走，每次只能向左下或右下，求路径和最大。
f[i][j] = max(f[i-1][j-1], f[i-1][j]) + a[i][j]
```

### 2.2 最长上升子序列 LIS（AcWing 895, 896）

**O(n²) 做法**：
```cpp
for (int i = 1; i <= n; i++) {
    f[i] = 1;  // 至少包含自己
    for (int j = 1; j < i; j++)
        if (a[j] < a[i])
            f[i] = max(f[i], f[j] + 1);
}
```

**O(n log n) 做法**：维护一个数组 q[]，q[len] = 长度为 len 的上升子序列的末尾最小值。

```cpp
int len = 0;
for (int i = 0; i < n; i++) {
    int l = 0, r = len;
    while (l < r) {
        int mid = l + r + 1 >> 1;
        if (q[mid] < a[i]) l = mid;
        else r = mid - 1;
    }
    len = max(len, r + 1);
    q[r + 1] = a[i];
}
```

### 2.3 最长公共子序列 LCS（AcWing 897）

```cpp
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++) {
        f[i][j] = max(f[i-1][j], f[i][j-1]);
        if (a[i] == b[j])
            f[i][j] = max(f[i][j], f[i-1][j-1] + 1);
    }
```

### 2.4 最短编辑距离（AcWing 902）

把字符串 a 变成 b 的最少操作次数（增/删/改）。

```cpp
// 初始化
for (int i = 0; i <= n; i++) f[i][0] = i;  // 删 i 次
for (int j = 0; j <= m; j++) f[0][j] = j;  // 插 j 次

for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++) {
        f[i][j] = min(f[i-1][j] + 1, f[i][j-1] + 1);  // 删/增
        if (a[i] == b[j]) f[i][j] = min(f[i][j], f[i-1][j-1]);      // 匹配
        else               f[i][j] = min(f[i][j], f[i-1][j-1] + 1);  // 替换
    }
```

## 三、区间 DP（AcWing 282）

### 特点

状态由**区间**定义，通常 `f[i][j]` 表示区间 [i, j] 的最优解。

### 石子合并

```cpp
// 枚举区间长度 len，再枚举左端点 i
for (int len = 2; len <= n; len++)
    for (int i = 1; i + len - 1 <= n; i++) {
        int j = i + len - 1;
        f[i][j] = INF;
        for (int k = i; k < j; k++)   // 枚举分割点
            f[i][j] = min(f[i][j], f[i][k] + f[k+1][j] + s[j] - s[i-1]);
    }
```

**三层循环**：长度 → 起点 → 分割点。`s[]` 是前缀和。

## 四、计数类 DP（AcWing 900）

### 整数划分

把 n 拆成若干正整数的和，求方案数。

**完全背包做法**：物品是 1~n，容量是 n，求方案数。

```cpp
f[0] = 1;
for (int i = 1; i <= n; i++)
    for (int j = i; j <= n; j++)
        f[j] = (f[j] + f[j - i]) % mod;
```

## 五、数位统计 DP（AcWing 338）

### 问题

统计 [a, b] 中每个数字 0~9 各出现多少次。

### 思路

`count(n, x)` = 1~n 中数字 x 出现的次数。答案 = count(b, x) - count(a-1, x)。

逐位分析：固定某一位为 x，分类讨论这一位前面的数和后面的数的取值范围。

这个比较复杂，建议直接背分类讨论的写法或者看 yxc 视频。

## 六、状态压缩 DP

### 核心思想

用一个**整数的二进制位**来表示一个集合/状态。

### 6.1 蒙德里安的梦想（AcWing 291）

在 N×M 的方格中放 1×2 的小方块，求方案数。

**状态**：`f[i][j]` = 前 i 列已经填好，第 i 列向第 i+1 列伸出的状态为 j 的方案数。

j 是一个 N 位二进制数，每一位表示该行是否有横放的方块伸到下一列。

### 6.2 最短 Hamilton 路径（AcWing 91）

**状态**：`f[S][j]` = 已经走过的点集为 S，当前在点 j 的最短路径长度。

```cpp
f[1][0] = 0;  // 起点是 0
for (int S = 0; S < 1 << n; S++)
    for (int j = 0; j < n; j++)
        if (S >> j & 1)  // j 在集合 S 中
            for (int k = 0; k < n; k++)
                if ((S ^ 1 << j) >> k & 1)  // k 在 S 去掉 j 后的集合中
                    f[S][j] = min(f[S][j], f[S ^ 1 << j][k] + w[k][j]);
```

**复杂度**：O(2^n × n²)，n ≤ 20 左右。

## 七、树形 DP（AcWing 285）

### 没有上司的舞会

每个人有一个快乐值，上下级不能同时参加。求最大快乐值总和。

**状态**：
- `f[u][0]` = 以 u 为根的子树中，u **不参加**的最大值
- `f[u][1]` = 以 u 为根的子树中，u **参加**的最大值

**转移**：
```
f[u][0] = Σ max(f[si][0], f[si][1])   // u 不去，子节点随意
f[u][1] = Σ f[si][0] + happy[u]        // u 去，子节点不能去
```

## 八、记忆化搜索（AcWing 901）

### 什么时候用？

当状态转移的**拓扑关系不明确**（不好确定遍历顺序）时，用记忆化搜索代替循环。

### 滑雪

从矩阵任意点出发，只能向更低的格子走，求最长路径。

```cpp
int dp(int x, int y) {
    if (f[x][y] != -1) return f[x][y];  // 已算过
    f[x][y] = 1;
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i], ny = y + dy[i];
        if (nx >= 0 && nx < n && ny >= 0 && ny < m && h[nx][ny] < h[x][y])
            f[x][y] = max(f[x][y], dp(nx, ny) + 1);
    }
    return f[x][y];
}
```

**本质**：DFS + 缓存 = 自顶向下的 DP。

## DP 总结

| 类型 | 状态定义 | 典型题 |
|------|---------|--------|
| 背包 | f[体积] = 最大价值 | 01/完全/多重/分组 |
| 线性 DP | f[位置] = 到此处的最优值 | LIS、LCS、编辑距离 |
| 区间 DP | f[左][右] = 区间最优值 | 石子合并 |
| 计数 DP | f[值] = 方案数 | 整数划分 |
| 数位 DP | 逐位分析 | 数字出现次数 |
| 状压 DP | f[集合] = 最优值 | Hamilton、棋盘覆盖 |
| 树形 DP | f[节点][选/不选] | 没有上司的舞会 |
| 记忆化搜索 | DFS + cache | 滑雪 |

### DP 做题口诀

> 先想**状态**怎么定义（集合+属性），再想**转移**怎么划分（集合怎么拆），最后处理**初始化**和**边界**。
