## 一、质数

### 1.1 判定质数（试除法）

**定义**：大于 1 的整数，只能被 1 和自身整除。

**关键优化**：如果 d 能整除 n，那么 n/d 也能整除 n。所以只需要枚举到 √n。

```cpp
bool is_prime(int x) {
    if (x < 2) return false;
    for (int i = 2; i <= x / i; i++)  // i <= x/i 等价于 i*i <= x，但不会溢出
        if (x % i == 0) return false;
    return true;
}
```

**为什么写 `i <= x / i` 而不是 `i * i <= x`？** 防止 `i * i` 溢出 int。

### 1.2 分解质因数（试除法）

**算术基本定理**：任何 > 1 的整数都能唯一分解为质数的乘积。

```cpp
void divide(int x) {
    for (int i = 2; i <= x / i; i++)
        if (x % i == 0) {
            int s = 0;
            while (x % i == 0) x /= i, s++;
            // i 出现了 s 次
        }
    if (x > 1) // x 本身是一个大于 √x 的质因子（最多一个）
}
```

**为什么 i 不需要判断是质数？** 因为当 i 是合数时，x 中 i 的因子已经在之前被除掉了。

### 1.3 筛法求素数

#### 朴素筛（埃氏筛）O(n log log n)

```
每个素数 p，把 p 的所有倍数标记为合数。
```

#### 线性筛（欧拉筛）O(n) ← 推荐

**核心保证**：每个合数只被它的**最小质因子**筛掉一次。

```cpp
for (int i = 2; i <= n; i++) {
    if (!st[i]) primes[cnt++] = i;
    for (int j = 0; primes[j] <= n / i; j++) {
        st[primes[j] * i] = true;
        if (i % primes[j] == 0) break;  // 关键：保证最小质因子筛
    }
}
```

**`if (i % primes[j] == 0) break` 的含义**：
- primes[j] 是 i 的最小质因子
- primes[j] * i 的最小质因子也是 primes[j]
- 如果不 break，后面的 primes[j+1] * i 的最小质因子仍然是 primes[j]（不是 primes[j+1]），会在以后被重复筛

## 二、约数

### 2.1 求所有约数（试除法）

和判素数一样，枚举到 √n 即可。

### 2.2 约数个数与约数之和

设 `N = p1^c1 × p2^c2 × ... × pk^ck`

```
约数个数 = (c1+1) × (c2+1) × ... × (ck+1)
约数之和 = (1+p1+...+p1^c1) × ... × (1+pk+...+pk^ck)
```

**直觉理解**：每个约数都是从每个质因子中选 0~ci 个的组合，约数个数就是选法总数。

### 2.3 最大公约数（欧几里得/辗转相除）

```cpp
int gcd(int a, int b) { return b ? gcd(b, a % b) : a; }
```

**原理**：`gcd(a, b) = gcd(b, a mod b)`，最终 b 变成 0 时 a 就是答案。

## 三、欧拉函数

### 定义

`φ(n)` = 1~n 中与 n 互质的数的个数。

### 公式

```
φ(n) = n × (1 - 1/p1) × (1 - 1/p2) × ... × (1 - 1/pk)
```

其中 p1...pk 是 n 的所有质因子。

**代码中写法**：`res = res / i * (i - 1)` 等价于 `res *= (1 - 1/i)`，先除后乘防溢出。

### 筛法求欧拉函数

在线性筛的过程中顺便求出每个数的欧拉函数值，利用三条性质：
1. p 是质数 → φ(p) = p - 1
2. i % p == 0 → φ(i×p) = φ(i) × p
3. i % p != 0 → φ(i×p) = φ(i) × (p - 1)

### 欧拉定理

若 a 与 n 互质，则 `a^φ(n) ≡ 1 (mod n)`

**费马小定理**（特例，n 是质数 p）：`a^(p-1) ≡ 1 (mod p)`

## 四、快速幂

### 核心思想

求 `m^k mod p`，把 k 拆成二进制：

```
m^k = m^(2^0 × b0) × m^(2^1 × b1) × ... × m^(2^30 × b30)
```

预处理 `m^1, m^2, m^4, m^8, ...`（每次自乘），哪些位是 1 就乘上。

### 模板

```cpp
int qmi(int m, int k, int p) {
    int res = 1 % p, t = m;
    while (k) {
        if (k & 1) res = res * t % p;  // 当前位是 1，乘上
        t = t * t % p;                  // 底数平方
        k >>= 1;                        // 右移一位
    }
    return res;
}
```

**时间复杂度**：O(log k)

### 应用：求逆元

若 p 是质数且 a 不是 p 的倍数，则 a 的逆元 = `a^(p-2) mod p`（费马小定理）。

## 五、扩展欧几里得算法

### 问题

求 x, y 使得 `ax + by = gcd(a, b)`

### 推导

当 b = 0 时：`a × 1 + 0 × 0 = a = gcd(a, 0)`，所以 x=1, y=0

递归：已知 `bx' + (a%b)y' = gcd(b, a%b) = gcd(a, b)`

展开 `a%b = a - ⌊a/b⌋×b`：
```
bx' + (a - ⌊a/b⌋×b)y' = gcd(a, b)
ay' + b(x' - ⌊a/b⌋×y') = gcd(a, b)
```
所以 `x = y', y = x' - ⌊a/b⌋×y'`

### 应用：线性同余方程

求 x 使得 `ax ≡ b (mod m)`，等价于求 `ax + my = b`（exgcd 求解）。

## 六、高斯消元

### 问题

解 n 元线性方程组。

### 步骤

1. 对每一列，找绝对值最大的行交换到当前行（数值稳定性）
2. 把当前行首位变成 1
3. 用当前行把下面所有行的这一列消成 0
4. 最后回代求解

### 三种结果

- `r == n`：唯一解
- `r < n` 且存在 `0 = 非零`：无解
- `r < n` 且全是 `0 = 0`：无穷多解

## 七、组合计数

### 四种求法，按数据范围选

| 方法 | 适用范围 | 时间 |
|------|---------|------|
| 递推 C[i][j] = C[i-1][j] + C[i-1][j-1] | a,b ≤ 2000 | O(n²) |
| 预处理阶乘+逆元 | a,b ≤ 10^5, p 是质数 | O(n log p) |
| Lucas 定理 | a,b 很大, p 较小质数 | O(p log p + log_p(n) × p) |
| 分解质因数+高精度 | 求真实值（无取模） | O(n) |

### 递推法（杨辉三角）

```
C(i,j) = C(i-1,j) + C(i-1,j-1)
```
就是上一行的两个数相加。

### 预处理逆元法

```
C(a,b) = a! / (b! × (a-b)!) = fact[a] × infact[b] × infact[a-b]
```
预处理 fact[] 和 infact[]，查询 O(1)。

### Lucas 定理

```
C(a,b) ≡ C(a%p, b%p) × C(a/p, b/p) (mod p)
```
把大数拆成 p 进制逐位算。

### 卡特兰数

```
Cat(n) = C(2n, n) / (n + 1)
```

**经典模型**：
- n 个 0 和 n 个 1 排列，任意前缀 0 的个数 ≥ 1 的个数
- n 个节点的不同二叉搜索树个数
- 合法括号序列个数

## 八、博弈论（简单）

### NIM 游戏

N 堆石子，两人轮流取（从任一堆取任意个），取完最后一个的人赢。

**结论**：先手必胜 ⟺ `A1 XOR A2 XOR ... XOR An ≠ 0`

**直觉**：异或为 0 是"平衡态"，先手怎么动都会打破平衡，后手总能恢复平衡。

### SG 函数

将任意公平组合游戏转化为图论问题：
- 每个局面是一个节点
- `SG(x) = mex({SG(y) | x 能到达 y})`
- mex = 不在集合中的最小非负整数

**定理**：
- 多个独立游戏的 SG 值 = 各子游戏 SG 值的 XOR
- 必胜 ⟺ SG > 0
